# 支票查詢系統 使用說明

## 系統需求
- Windows 10 / 11
- 網際網路連線（首次使用需下載 OCR 模型，約 100MB）
- AD 網域帳號需有共用資料夾的讀寫權限

---

## 安裝步驟（IT 人員執行一次即可）

### 方法一：使用安裝包（推薦）
1. 在有 Python 環境的電腦上執行 `build.bat`
2. 等待約 5 分鐘打包完成
3. 將 `dist\支票查詢系統.exe` 複製到各財會電腦

### 方法二：直接執行 Python
1. 安裝 Python 3.10+（https://python.org）
2. 執行 `build.bat` 安裝套件
3. 之後可用 `run.bat` 直接啟動

---

## 共用資料夾結構

```
\\server\share\支票\          ← 共用資料夾根目錄
├── 20260101.pdf              ← 支票掃描 PDF（直接放根目錄）
├── 20260320.pdf
├── ...
└── index\                   ← 系統自動建立，請勿刪除
    └── master_index.json
```

若 PDF 放在子資料夾，可在設定中填入子資料夾名稱。

---

## 使用流程

### 第一次使用
1. 開啟程式 → 出現「歡迎」提示
2. 填入共用資料夾路徑（例：`Z:\支票` 或 `\\server\share\支票`）
3. 點選「確定」→ 系統詢問是否建立索引 → 點「是」
4. 等待索引建立（500張支票約需 10-20 分鐘）
5. 完成後即可開始查詢

### 日常查詢
1. 在搜尋框輸入支票號碼（如 `HF1377789`）
2. 按 Enter 或點「搜尋」
3. 在結果列表選取目標支票
4. 右側出現預覽圖
5. 點「列印支票」→ 確認預覽 → 列印

### 新增 PDF 後
- 選單 → 功能 → **更新索引（只掃新增 PDF）**（較快）
- 或選 **重新建立索引（全部）**（完整重建）

---

## 搜尋技巧

| 輸入方式 | 說明 | 範例 |
|---------|------|------|
| 完整號碼 | 精確搜尋 | `HF1377789` |
| 部分號碼 | 模糊搜尋（含此字串） | `1377` 可找到 `HF1377789`、`AB13770` |
| 數字部分 | 模糊搜尋 | `789` 可找所有結尾是789的支票 |

## 常見問題

**Q: 第一次開啟很慢？**  
A: 首次使用需要下載 OCR 模型（約 100MB），需要等待。之後開啟就很快了。

**Q: 找不到某張支票？**  
A: 可能原因：
1. 該 PDF 尚未索引 → 執行「更新索引」
2. OCR 辨識失敗（如圖片模糊） → 可在「進階」查看未辨識頁面
3. 支票號碼格式不符 → 在設定中調整「支票號碼格式」

**Q: 列印時比例不對？**  
A: 請在印表機設定中選擇「符合頁面」或「實際大小」，不要選「縮放」。

**Q: 如何判斷一頁一張還是兩張？**  
A: 系統會自動偵測。若上下各找到一個支票號碼，即視為一頁兩張；只找到一個則視為一頁一張。

## 技術資訊

- OCR 引擎：EasyOCR（本機處理，不上傳任何資料）
- PDF 處理：PyMuPDF
- GUI：PyQt6
- 索引格式：JSON（明文可讀）
- 設定儲存：`%APPDATA%\CheckFinder\config.json`（每台電腦各自）
